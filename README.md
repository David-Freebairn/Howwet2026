Howwet uses a daily water-balance model (HowLeaky)  which describes the daily dynamics of water in agricultural systems based on weather and soil type . Processes include infiltration, runoff, evaporation and deep drainage to track plant available soil water.
Parameters describing soil properties are retrieved from a small database of representative soils.
